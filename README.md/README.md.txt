# 🍽️ Southern Bites - Restaurant Website

A simple yet elegant **restaurant website** built with **HTML5 + CSS3**.  
Designed to showcase an **authentic Southern dining experience** with both vegetarian and non-vegetarian options.  

👉 [Live Demo](https://MEGATRON-X-49.github.io/southern-bites-website/)  
👉 [Source Code](https://github.com/MEGATRON-X-49/southern-bites-website)

---

## ✨ Features
- Responsive multi-page design (Home, Menu, About, Contact)  
- Smooth scrolling navigation  
- Minimal, modern UI with custom backgrounds  
- Menu highlights both veg 🌱 and non-veg 🍗 dishes  
- Contact form for inquiries  
- Hosted with GitHub Pages for live demo
